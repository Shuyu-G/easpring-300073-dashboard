# 股市预测

这个仓库现在作为多股票项目的根目录使用。

## 当前目录结构

- `.github/`
  仓库级 GitHub Actions 与 GitHub Pages 部署配置
- `.venv/`
  本地共用 Python 虚拟环境
- `当升科技/`
  北京当升材料科技股份有限公司 (`300073`) 的完整 ETL、模型、Dashboard、数据与报告

## 当前已接入的股票

- [当升科技/README.md](当升科技/README.md)

## 继续新增另一只股票时的建议

1. 在根目录下新建一个股票子目录
2. 复制 `当升科技/` 的结构作为模板
3. 修改该子目录里的公司名、股票代码、数据源关键词、Dashboard 文案
4. 再决定 GitHub Pages 默认发布哪一只，或者改成股票切换首页
